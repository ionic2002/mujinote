# dust.js 에서 중요한 핵심 문법

- {?foo} === true
- {^foo} === false
- {#foo} === o.hasOwnProperty('foo')
- {@select key=foo} {@eq value="var"} === switch(foo) { case 'var': }

되도록 4가지내에서 처리 가능하도록 데이터 수정 

## true

- String
- Number
- {}
- true

## false

- null
- undefined
- “”
- ‘’
- []
- false
